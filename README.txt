BLAST.java


USAGE: BLAST <Fasta formatted file of database sequences> <Fasta formatted file of query sequences>

BLAST is a local alignment tool. This is an implementation of an older version of BLAST
It will find database hits for a given query sequence file. 

This version also has information about the runtimes. 

It requires IndexPair.java and Hit.java to run. 

Credit to Amanda Hartung for showing me how to format the output.
